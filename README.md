# HerLog AI

HerLog AI is a memory-driven health assistant designed to track and understand recurring symptoms over time.

## Problem
Most health tools are stateless. They log symptoms but fail to connect patterns across time, making it difficult for users to understand recurring health issues.

## Solution
HerLog AI uses a memory-based approach to track symptoms like irritation and identify patterns over time, including persistence and cycle-based triggers.

## Key Features
- Tracks recurring symptoms (irritation/itch)
- Detects persistence over time
- Identifies menstrual cycle patterns
- Improves responses using memory

## How It Works
User Input → AI Processing → Memory Retrieval (Hindsight) → Improved Response

The system stores past interactions and retrieves relevant ones to generate better, context-aware responses.

## Tech Stack
- Python (Flask backend - conceptual)
- HTML/CSS/JS (frontend - conceptual)
- Hindsight (memory layer)
